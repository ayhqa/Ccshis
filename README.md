## 智选云开发平台 ##

**智选云开发平台** 基于ddd(领域驱动设计)+cqrs(命令也查询职责分离开)+micservice(微服务)的开发框架，在使用该架构时，你需要提前了解以上开发方式.

### 架构特点 ###

- 微服务
- 分布式
- 高并发
- 横向扩容
- saas平台化
- iot物联网化
- 容器化
